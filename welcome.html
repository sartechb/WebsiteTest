Wobetto helps you find study groups to work with and lets create your own groups. You can filter these groups by searching for classes that you're in. Click the button below to start your search.

Let's start by adding your classes as filters. Click on a filter to turn it on. Active class filters show up blue. Inactive ones will be gray.

Don't worry if a class isn't available. It just means no one has posted about that class yet. You can add more class filters later.

Wobetto also allows you to choose your work space. You can filter these groups by searching for the location that you're in. Click the button below to start your search.

Wobetto allows you to customize your searches by turning on different filters in any combination, so you find groups for the classes that matter in places that matter.

In additon to class and location filters, you can also create phrase filters.

Use this box to create posts for new study groups. In the app, clicking on it will open up a blog-style layout that allows you to specify various things, such as classes, where you’re working, and how many people you’d like to join you.
